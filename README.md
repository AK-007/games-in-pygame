# Games in PyGame

### Installation:
Requires Python2

Install dependencies by running `pip install -r requirements.txt`

## Asteroides:

### Usage:
Run `python asteroides.py` to start the game.

Use arrow keys to move up, down, left and right on the canvas.  
Use space bar to shoot.
